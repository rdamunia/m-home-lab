# m-home-lab

## 项目简介

`m-home-lab` 是一个用于归档与发布多个独立 HTML 页面的仓库。  
这里存放的页面均为单个 HTML 文件，彼此独立，不依赖特定域名或后端服务。  
仓库本身不针对任何具体网站，仅作为页面存储与版本管理的载体。

## 目录说明

```
m-home-lab/
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 页面引用的静态资源（图片、样式、脚本等）
└── README.md     # 本文件
```

- **pages/**：每个子文件夹或独立文件对应一个归档页面，命名以页面功能或标识为主。
- **assets/**：按需组织，资源文件与对应页面保持关联，便于维护。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可独立打开并运行。
- 页面内容与形式可能随时间更新，历史版本可通过 Git 提交记录追溯。
- 归档页面不包含任何外部跟踪、广告或用户数据收集功能。
- 部分页面可能引用外部资源（如字体、图标库），但均不涉及具体服务承诺。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issues 提出建议或报告问题，但不保证即时响应。
- 不接受直接向 pages/ 目录提交与归档无关的页面或内容。
- 如需引用或分发本仓库中的页面，请保留原始文件完整性，并注明出处。
